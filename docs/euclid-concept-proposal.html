Overview

This proposal documents the code classification strategy for 4335 Euclid Avenue. The building is designed with individual private entries for every residential unit — each upper-floor unit has its own private stair entry at grade, and each ground-floor unit has its own direct exterior door. Each stacked pair of units is enclosed on both sides by a full-height fire wall, forming self-contained residential unit blocks.

That design — private non-shared egress per unit plus CBC 706 fire walls — is what qualifies the building for Group R-3 classification rather than R-2. R-3 unlocks a lighter fire sprinkler system (residential NFPA 13D instead of the commercial-grade NFPA 13R), reduces ongoing maintenance obligations, and produces a building that feels more like individual townhomes than a conventional apartment complex.

✓
Key outcome: No shared hallway. No shared stairwell. Every unit has its own front door at street level. Each unit — including upper-floor units — is entered independently from the exterior. Residents never pass through another unit's space to reach their home.

Code Basis (Summary)

The following code sections form the basis for the R-3 classification and sprinkler system selection. Full code analysis will be submitted on the permit drawings.

Code Section Subject Application Result
CBC 706 Fire Walls Full-height fire walls, structurally independent, foundation through roof sheathing, between every unit block ✓ Each block = separate building
CBC 310.5 Group R-3 Classification Buildings with not more than 2 dwelling units — each fire-wall-separated block contains exactly 2 units ✓ R-3 applies to each block
IB-125 Ex. 4 DSD Classification Table — City of San Diego "New Townhouses Designed per CRC" → R-3, Combination Permit. City of San Diego's own Information Bulletin 125 (June 2025) explicitly confirms this project type as R-3. This project matches Example 4 exactly. ✓ DSD confirms R-3 for new CRC townhouses — strongest direct authority
CBC 903.2.8 NFPA 13R Sprinkler Mandate Applies to Group R-2 only — not triggered for R-3 ✓ NFPA 13R not required
CRC R313 Residential Sprinkler Requirement New residential construction requires automatic sprinkler system — NFPA 13D acceptable for R-3 ✓ NFPA 13D required (lighter system)
CBC 508.4 B / R-3 Occupancy Separation Fire-rated separation between commercial block (Group B) and residential blocks (Group R-3) ✓ Shown at commercial/residential interface
⚠
Note on Sprinklers NFPA 13D is required — it is not eliminated entirely. New residential construction in California requires some form of automatic sprinkler system. The benefit of this design is upgrading from the more expensive NFPA 13R (commercial-grade, full-building system) to NFPA 13D (residential-grade, per-unit system), which is smaller in scope, cheaper to install, and simpler to maintain.

Cost Savings — Fire Wall Strategy vs. Full Sprinkler System

The R-3 fire wall design replaces the NFPA 13R commercial-grade system with NFPA 13D residential-grade. The fire wall assembly (USG SA-925 Solid System B) is an added cost, but the sprinkler system savings offset it — and the R-3 classification delivers additional long-term benefits that go beyond construction cost.

Item R-2 Path (No Fire Walls) R-3 Path — Proposed (CBC 706 Fire Walls)
Fire Wall Assembly Not required USG SA-925 Solid System B — 2-hr, 3½" core, STC 54–60 (THERMAFIBER SAFB). 4 walls × ~500 SF each. Est. ~$24,000–$40,000 installed
Sprinkler System Type NFPA 13R — Commercial-grade, full building NFPA 13D — Residential-grade, living spaces only
Sprinkler Coverage All areas incl. corridors, utility spaces, attic Living spaces only — attic and garage excluded per NFPA 13D
Pipe Sizing Larger diameter, higher flow demand Smaller diameter, residential flow rates
Water Meter May require dedicated fire service meter Can share domestic meter in most cases
Sprinkler Installation (9 units) ~$45,000 – $65,000 ~$18,000 – $32,000
Total Fire Protection Cost ~$45,000 – $65,000 ~$42,000 – $72,000 (fire walls + 13D)
Net Cost Delta Roughly cost-neutral at best case; R-3 path breaks even or saves up to ~$23,000 — sprinkler savings offset fire wall cost
⚠
These are planning-level estimates only. Final cost depends on building layout, pipe routing, and the C-16 contractor's scope. A sub quote for NFPA 13D scope and a USG-certified framing sub quote for SA-925 should be obtained before permit submittal to validate the financial case.
✓
Why the R-3 path wins beyond construction cost: Even at cost-neutral, the R-3 path delivers: (1) R-3 classification — simpler permit, no CBC 903.2.8 NFPA 13R mandate; (2) No shared corridor to build or maintain; (3) Private entries per unit — higher desirability and resale value; (4) 13D lower ongoing maintenance cost; (5) STC 54–60 party wall sound isolation included in the SA-925 assembly at no added cost over a plain wall.

Risks & Fallback

The R-3 fire wall strategy is well-supported by code and by DSD's own IB-125 Table 1 (Example 4). However, it is confirmed at plan check — not at a pre-application meeting. The following risks should be understood before submittal.

Risk Likelihood Impact Mitigation
Plan check rejects R-3 classification — classifies building as R-2 Low–Med NFPA 13R required → ~$20K–32K additional cost; shared egress redesign Lead permit set with IB-125 Ex. 4 citation + CBC 310.5 + CBC 706 code analysis sheet. Non-shared egress clearly shown on all plans.
Fire wall structural independence detail challenged Medium Correction letter → design revision → resubmittal delay (4–8 weeks) Provide complete CBC 706.4 structural independence detail at permit submittal. Show independent footing, wall framing, and roof connection on structural sheets.
San Diego Fire-Rescue requires 13R instead of 13D Low Increased sprinkler system cost (~$20K–32K delta) Confirm NFPA 13D acceptance for new R-3 townhouses at this address with Fire-Rescue before permit submittal. Site is not in a Fire Hazard Severity Zone.
✓
Fallback position is well-defined. If R-3 classification is denied at plan check, the project reverts to R-2 with NFPA 13R. The floor plan layout (private entries, fire walls) is still a better building — only the sprinkler system scope changes. No redesign of the unit layout is required.
