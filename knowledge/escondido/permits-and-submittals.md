Source: City of Escondido Development Services Dept — Building Division.
(760) 839-4647 | https://www.escondido.org/building

## 1. Permit Types

| Permit Type | Trigger | Notes |
|-------------|---------|-------|
| Building Permit — New Construction | New SFD, ADU, commercial | Full plan check |
| Building Permit — Addition/Alteration | Additions > 120 sf, structural changes | Plan check |
| Mechanical / Electrical / Plumbing | System-specific work | May be OTC |
| Grading Permit | Cut/fill > 50 cy; hillside development | Engineering review |
| Demolition Permit | Full or partial demolition | Asbestos survey if applicable |
| Solar PV Permit | PV installation | OTC per SB 379 |
| Fire Sprinkler Permit | NFPA 13/13D installation | EFD — deferred |
| Right-of-Way / Encroachment | Work in public ROW | Engineering |
| Hillside Development Permit | Average slopes ≥ 15% | Planning + Engineering |

## 2. Plan Check Submittal Requirements

### Residential New Construction (SFD / ADU)

1. **Site Plan** (≤ 1 in = 20 ft) — property lines, setbacks, easements, topo (required for
   any hillside lot), drainage arrows, utilities, North arrow
2. **Floor Plans** — dimensioned, room labels, door/window schedule
3. **Elevations** — all four faces, materials labeled
4. **Sections** — minimum one building section
5. **Foundation Plan** — slab/footing, soils bearing, geotechnical reference if required
6. **Structural Plans + Calcs** — if engineered (required for hillside, irregular, or > 2-story)
7. **Roof Plan** — drainage, solar rough-in
8. **Title 24 Energy CF1R** — from CEC-approved software
9. **CALGreen Residential Checklist** — mandatory measures
10. **C&D Waste Management Plan** — ≥ 65% diversion
11. **Stormwater BMP Site Plan** — standard project BMPs
12. **Hillside Analysis** — slope analysis map and cut/fill calculations if applicable

### Over-the-Counter (OTC)

- Solar PV per SB 379 streamlined.
- Simple alterations (no structural, no new bedrooms).
- Trade permits for standard replacements.

## 3. Plan Check Process

```
1. Pre-application (optional) — recommended for hillside, ADU, new SFD
2. Submittal — counter or online portal
3. Intake + fee payment
4. Cycle 1 — ~15–25 biz days (SFD/ADU); ~25–40 biz days (MF/commercial)
5. Correction list issued
6. Applicant response + resubmittal
7. Cycle 2 — ~10–15 biz days
8. Permit ready — pay balance, school fees
9. Permit issued — post on site
10. Inspections
11. Final inspection → Certificate of Occupancy
```

---

## 4. Required Inspections

| Inspection | Timing |
|------------|--------|
| Foundation / Pre-pour | After forms, rebar set |
| Underground Plumbing | Before backfill |
| Frame | All framing exposed |
| Insulation | Before drywall |
| Drywall Nailing | Before finishing |
| Rough Trades | During frame stage |
| Final Building / Trades / Fire | All work complete |

  Contact EUSD (760) 432-2400 and EUHSD (760) 291-3200.
