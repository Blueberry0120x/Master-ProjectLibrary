# City of Escondido — Plan-Check Reference Library

Master jurisdictional reference for City of Escondido permit submittals.
Scope: Single-family (SFD), ADU/JADU, additions, accessory structures, and small commercial.
Located in northern San Diego County, CA.

Code cycle baseline: **2022 California codes** (CBC/CRC/CPC/CMC/CEC/CALGreen/CFC)
as locally amended per Escondido Municipal Code (EMC).
**2025 cycle effective Jan 1, 2026.**

  - **Title 33** — Zoning Code (zones, overlays, development standards, ADU)

## Global Notes / Gotchas

- **WUI exposure.** Unlike Garden Grove or Santa Ana, Escondido has significant
  wildland interface areas. Many parcels in north/east Escondido are in VHFHSZ.
  Always check the CAL FIRE FHSZ map before designing — Chapter 7A materials
  requirements add significant cost.
- **Hillside development.** Escondido has significant hillside terrain (Elfin Forest,
  Harmony Grove, Lake Wohlford area). Hillside development ordinance triggers
  additional grading limits, slope analysis, and geotechnical requirements.
- **Water District.** Most of Escondido is served by Vallecitos Water District — not
  the City. Capacity fees and will-serve letters come from Vallecitos, not City PW.
- **Agricultural overlay.** Some eastern parcels have agricultural zoning — ADU
  rules may differ; verify zone carefully.
- **2025 code cycle.** Verify with Building Division which cycle applies.
