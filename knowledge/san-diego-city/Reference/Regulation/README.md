# City of San Diego — Plan-Check Reference Library

Master jurisdictional reference for City of San Diego permit submittals.
Scope: Single-family (SFD), two-family, multifamily (R-2), ADU/JADU, mixed-use,
and small commercial where it intersects residential work.

Code cycle baseline: **2022 California codes** (CBC/CRC/CPC/CMC/CEC/CALGreen/CFC)
as locally amended. **2025 code cycle takes effect Jan 1, 2026** — projects
submitted on/after that date must comply with the new adoption (CBC/CFC/CWUIC
local-amendment package issued late 2025). Flag any ambiguity on your plans.

Active project context: **7201 Garden Grove Blvd, STE A, San Diego 92841** — PRJ 1142530,
detached ADU, 2-story, 1,200 sf, wood-frame, stucco. RS zone (verify on Zoning
& Parcel Information portal). Review this file plus `adu-and-sb9.md` and
`waste-and-calgreen.md` before touching the correction letter.

## Primary Code References

- **San Diego Municipal Code (SDMC)** — codified city ordinance, organized by
  Chapter/Article/Division. Mirror: docs.sandiego.gov/municode
- **Land Development Code (LDC)** = SDMC Chapters 11–15 (General Plan/Zoning/
  Subdivision/General Regs/Procedures).
- **Chapter 11** — Land Development Procedures
- **Chapter 12** — Land Development Reviews (incl. environmental)
- **Chapter 13** — Zones (Base, Overlay, Planned District)
- **Chapter 14** — General Regulations (parking, signs, landscape, density
  bonus, ADU, Complete Communities, stormwater)
- **Chapter 15** — Planned Districts
- **Chapter 9** — Building Regulations (local amendments to CBC/CRC/CEC/CPC/CMC)
- **Chapter 5 Article 11–12** — Fire Code + WUI local amendments (CFC/CWUIC)
- **Chapter 6 Article 6** — Landscape & MWELO
- **Chapter 6.5** / **Ch 66 Art 7** — Recycling, C&D Ordinance (SDMC 66.0702 ff.)

## Global notes / gotchas

- **Project number vs permit number.** `PRJ-xxxxxxx` = project container;
  `PMT-xxxxxxx` = the issued permit; `APR-xxxxxxx` = approval/plan set.
  Correction letters cite PRJ + Approval. WMF wants the PMT (may be blank at
  intake — okay to note "pending issuance").
- **Community Plan Areas** can add restrictions beyond SDMC (e.g., City
  Heights, Uptown, Clairemont Mesa, La Jolla, Barrio Logan). Always confirm
  the community plan for the parcel.
- **Overlays stack.** Coastal, Airport, Parking Impact, Transit Area,
  Residential Tandem Parking, MHPA, Hillside, Sensitive Biological Resources —
  a single parcel can have four or five.
- **Historic resources.** Parcels > 45 years old trigger historical review;
  individually designated landmarks block SB 9 and many CC programs.
- **Corporate network note** (Stantec): DSD and docs.sandiego.gov are
  reachable; OpenDSD may need HTTPS cert bypass.
