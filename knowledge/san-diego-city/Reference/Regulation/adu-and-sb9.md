# ADU, JADU, and SB 9 — City of San Diego

Primary local ordinance: **SDMC §141.0302** (Accessory Dwelling Units) for
ADU/JADU and **SDMC §143.1303** for SB 9 two-unit developments + urban lot
splits. State floor: Gov Code §65852.2 (ADU) and §65852.21 + §66411.7 (SB 9).

Current bulletin: **Information Bulletin 400 — Accessory Dwelling Unit /
Junior Accessory Dwelling Unit** (Aug 2025 edition — incorporates Ordinance
O-21989 effective Aug 22, 2025, which revised setback rules; supersedes Nov
2024 edition). SB 9 / multi-unit lot split guidance: **IB-409**.

---

## ADU — size, height, setbacks

### Size caps (detached ADU)

| Unit type | Max size |
|-----------|----------|
| **Detached ADU — state floor** | 1,200 sf (cannot be reduced by City) |
| **Attached ADU** | 50% of primary dwelling's living area OR 1,200 sf, whichever is less (state minimum 800 sf always allowed) |
| **Conversion ADU** (existing space + up to 150 sf ingress/egress) | No additional cap beyond existing envelope + 150 sf |
| **JADU** | **≤ 500 sf** maximum (no state minimum floor area); within existing or proposed SFD footprint (or attached garage); shared kitchen/bath allowed with primary |

City allows detached ADU > 1,200 sf if the overall lot FAR permits — treated
as a second dwelling unit under base-zone rules. Most owners stay at 1,200 sf
to retain ministerial state ADU preemption.

### Height limits (state + City, per IB-400)

| Condition | Max height |
|-----------|-----------|
| Detached ADU on SFR lot (default) | **16 ft** |
| Detached ADU within ½ mile walking of major transit stop OR in an Urban Lot Split area | **18 ft** (one story) |
| Detached ADU on a lot with multifamily dwelling (≥ 3 units) | **18 ft** |
| Attached ADU to 2-story primary | **25 ft** (or height of primary, whichever is less) |
| ADU in MF multistory building (existing) | Same as primary |

**August 2025 amendment:** City limited detached ADUs in single-family zones
to **two stories maximum**. Deeper three-story detached ADUs that were
allowed under the Bonus ADU program pre-amendment are no longer permitted
in RS zones (verify grandfathering with DSD).

### Setbacks (SDMC 141.0302(b)(9))

| Location | Setback |
|----------|---------|
| **Front** | Per base zone (no ADU exception) |
| **Street side** | 4 ft OR base-zone street-side setback, whichever is less |
| **Interior side (detached ADU ≤ 16 ft tall)** | **0 ft** — San Diego local code exceeds state minimum (Ordinance O-21989, effective Aug 22, 2025) |
| **Rear (detached ADU ≤ 16 ft tall)** | **0 ft** — same as above |
| **Interior side/rear (detached ADU > 16 ft or > 1 story)** | **4 ft** minimum (Ordinance O-21989; this is the rule for the active 2-story project at 5552 Redwood) |
| **Conversion ADU** | May maintain the existing structure's setbacks even if nonconforming (even 0 ft) — Gov Code §65852.2(a)(1)(D)(vii) |
| **Between buildings on same lot** | Per CRC / CBC fire separation; typically 6 ft or 1-hr rated wall |

**Pre-Aug 2025 rule (archived):** Before Ordinance O-21989, the file incorrectly
stated 4 ft for ≤16 ft ADUs. State law sets a "no more than 4 ft" ceiling on
what localities can require; San Diego went further and eliminated the setback
for ≤16 ft structures. The 4 ft rule now applies only to taller units.

### Fire sprinklers (IB-400 Aug 2025 + SDFD Technical Bulletin RESD 3-4)

**Standard waiver rule:** Detached ADU ≤ 1,200 sf is **not required** to have
sprinklers if the primary dwelling does not have them (Gov Code §65852.2 +
IB-400). This is a state preemption — San Diego cannot override it for
standard ADUs.

**Tie-in rule:** If the primary IS sprinklered, the ADU must connect to the
sprinkler system or install its own NFPA 13D system.

**Mandatory sprinklers regardless of primary:**
- ADU in VHFHSZ — confirm with SDFD Community Risk Reduction (may be
  required even if primary is unsprinklered)
- **Bonus ADU** (deed-restricted affordable under SDMC 141.0302(d)) — sprinklers always required
- **Affordable ADU** under Density Bonus program — sprinklers always required

**Construction type does NOT affect the waiver.** Whether the ADU is built
to Type V-A (1-hr rated) or Type V-B (0-hr) has no bearing on the sprinkler
waiver. The V-A 1-hr rating is a structural fire-resistance requirement;
CRC R313 sprinklers are a separate, independent requirement triggered by
dwelling type and the primary's sprinkler status only.

**JADU:** Same waiver — no sprinklers required if primary is unsprinklered.
If primary is sprinklered, JADU must be included in the system.

**Decision table:**

| ADU type | Primary unsprinklered | Primary sprinklered | VHFHSZ |
|---|---|---|---|
| Detached ≤ 1,200 sf | No sprinklers | Tie-in required | Confirm SDFD |
| Attached | No sprinklers | Tie-in required | Confirm SDFD |
| JADU | No sprinklers | Include in system | Confirm SDFD |
| Bonus / Affordable ADU | Required (program rule) | Required | Required |

Reference: SDFD Technical Bulletin RESD 3-4 (sprinklers for ADUs) and
IB-400 (Aug 2025). VHFHSZ map: sandiego.gov/fire/community-risk-reduction.

## Bonus ADU Program (SDMC 141.0302(d))

San Diego's local bonus program layered on top of state ADU law (**not** to
be confused with state ADU-within-density-bonus or state AHDB):

- Applies in **Transit Priority Areas (TPA)** only.
- Owner can build **one deed-restricted affordable (moderate-income) ADU**
  and receive **one unrestricted bonus ADU** as reward — so 2 ADUs instead
  of 1.
- Deed-restricted for **15 years** rental at moderate-income rent limits.
- In **Sustainable Development Area (SDA)** (CAP-defined), the bonus could
  scale higher (1 restricted → 2 bonus) up to a cap.

Post-**August 2025** amendment: the Bonus ADU program was significantly
tightened — verify current unit cap and restrictions at DSD before
designing a project around bonus ADUs. The two-story detached ADU limit
reduced the previous 3–4 story bonus designs.

## Parking for ADU

Zero required if ANY:
- Within ½-mile walking of transit (includes all TPAs).
- On-street permit district prohibits new residential permits.
- Historic district.
- Conversion of existing garage or interior space.
- Within one block of car-share.

Otherwise: **1 off-street space** (may be tandem in driveway or in setback).
Replacement parking for converted garage **not** required.

---

## Impact fees / school fees

- ADUs **< 750 sf**: exempt from Development Impact Fees (DIF) by state law.
- ADUs **≥ 750 sf**: DIF charged proportionally to primary dwelling's floor
  area.
- School impact fees apply to ADUs ≥ 500 sf per SDUSD policy (verify Level 1
  rate).

---

## Utilities

- ADU may share primary service OR have separate meter.
- **Separate water meter required** if lot creates new primary (SB 9
  split); optional for ADU.
- Sewer capacity fee applies to ADUs ≥ 500 sf (reduced rate).

## SB 9 — Two-unit development + urban lot split (SDMC §143.1303)

Incorporated into City code as **Multi-Dwelling Unit and Urban Lot Split
Regulations for Single-Family Zones** — a ministerial pathway parallel to
the City's conventional discretionary route.

### Eligibility

| Requirement | Threshold |
|-------------|-----------|
| Zoning | RS, RE, RX, RT and Planned District zones that **allow only single-family development** |
| Minimum lot (pre-split) | **≥ 2,400 sf** |
| Each resulting lot (split) | **≥ 1,200 sf**, **≥ 40%** of original parcel area |
| Total DU allowed | **2 DU per resulting lot** (4 DU total after lot split) |
| Plus ADUs? | Yes — up to 2 ADUs on non-split SB 9 two-unit projects |

### Exclusions

- Prime farmland, conservation, habitat areas.
- Hazardous waste sites.
- **Individually listed historic landmarks** (per AB 1061 amendment —
  historic **districts** no longer block SB 9).
- MHPA (Multi-Habitat Planning Area) of MSCP.
- Projects that would require demolition of > 25% of exterior structure of
  existing rental or deed-restricted housing.

### Setbacks (SB 9 units)

- Single-story units ≤ 16 ft: **no** rear/interior side setback required.
- Two-story or > 16 ft: base-zone setbacks.
- Front: base zone.

### Owner occupancy (urban lot split only)

- Applicant must sign a 3-year owner-occupancy affidavit on one resulting
  parcel (AB 1033 did not disturb this).

### Short-term rentals

- **Prohibited** on SB 9 lots (state law restricts to ≥ 30-day rentals).

## Quick reference table — ADU vs SB 9 vs primary

| Feature | Primary SFD | ADU (detached) | SB 9 unit | JADU |
|---------|-------------|----------------|-----------|------|
| Max DU | 1 per lot | 1 (+1 bonus in TPA) | 2 per resulting lot | 1 |
| Max size | Per FAR | 1,200 sf (state cap) | ~800 sf min guaranteed | 150–500 sf |
| Height | 30 ft + projections | 16 ft / 18 ft TPA / 25 ft attached | 16 ft (to skip setback) | Within primary |
| Min side setback | Per zone (typ 4–10 ft) | 4 ft | 0 ft (if ≤ 16 ft) | N/A — inside SFD |
| Parking min | 1–2 | 0 or 1 | 0 or 1 | 0 |
| Ministerial | No (discretionary if in overlay) | Yes (if within state preemption) | Yes | Yes |
| Rentable long-term | Yes | Yes | Yes | Yes (with owner-occupancy) |
| Short-term rental | Per STRO ordinance | Per STRO | **No** | No |
| Sprinkler | 13D required | Not required if ≤ 1,200 sf and primary isn't | 13D if new SFD scale | Primary's system |

## Checklist — ADU submittal (IB-400 aligned)

- [ ] Site plan w/ setbacks, BMZ, utilities (IB-122)
- [ ] Floor plans (primary + ADU), elevations, sections, roof plan
- [ ] Structural plans + calcs (or prescriptive IRC compliance)
- [ ] Title 24 CF1R + CALGreen residential checklist
- [ ] WMF + C&D deposit estimate
- [ ] Water/sewer service note (shared OK; separate meter if requested)
- [ ] Stormwater intake form (usually Standard Project tier)
- [ ] Fire — address numbering, sprinkler note, setback verification
- [ ] Parking — note exemption (TPA, conversion, etc.) or show space
- [ ] Deed-restriction form if claiming Bonus ADU
