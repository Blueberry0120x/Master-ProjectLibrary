# Permits & Submittal Workflow — City of San Diego DSD

## Permit families

| Permit | What it covers | Trigger |
|--------|----------------|---------|
| **Combination Building Permit** | SFD, duplex, ADU/JADU, and sometimes townhome — bundles bldg + electrical + plumbing + mechanical in one review | Most residential work |
| **Building — General / All Others** | Multifamily (R-2), commercial, industrial, mixed-use | > duplex, or commercial |
| **Trade Permits** | Stand-alone Electrical, Plumbing, Mechanical | Repair/replace, no structural |
| **Construction Change (CC)** | Revisions to an already-issued permit | In-construction design changes |
| **Deferred Submittal** | Trusses, fire sprinklers/alarm, solar, racking, shop drawings | Called out on approved plans |
| **Grading Permit** | Earthwork ≥ 200 cy or in overlay | SDMC Ch 14 Art 2 Div 2 |
| **Right-of-Way / Encroachment** | Work in public ROW, sidewalks, driveways | Separate from building |
| **Coastal Development Permit (CDP)** | Work in Coastal Overlay | SDMC Ch 12 Art 6 Div 5 |
| **Neighborhood Development Permit / Site Development Permit** | Discretionary — BMZ variances, ESL, sensitive resources | SDMC Ch 12 Art 6 |
| **Master Use Permit** | Large or phased development | — |

## Workflow — residential (Combination) submittal

```
1. Pre-App (optional)          ← Sched. via DSD; recommended for complex
2. Accela intake               ← aca-prod.accela.com/SANDIEGO → upload PDFs
3. PRJ + Invoice issued        ← pay initial plan-check fee
4. Intake screening (1–3 biz days) ← completeness check only
5. Discipline assignment       ← Building, Zoning, Fire, Stormwater, etc.
6. Cycle 1 review (~20–40 biz days for SFD, 30–60 for MF/commercial)
7. Cycle issues report         ← ONE consolidated list from all disciplines
8. Applicant response          ← narrative + revised plans + resub
9. Cycle 2 review (~10–20 biz days)
10. Approval (APR#) → permit ready to issue
11. Pay balance + school fees + impact fees + WMF deposit
12. PMT# issued → inspections begin
```

Reviewers issue their discipline reports independently, but resubmittal is
**consolidated** — applicant cannot resubmit to one discipline at a time.
All disciplines must finish a cycle before the applicant uploads response.

Typical cycle count: **2–3 cycles** for a clean ADU/SFD; **3–5** for infill
MF or commercial. Deposit-to-invoice fee model — more cycles = more fees.

## Over-the-Counter (OTC) programs

- **OTC Ministerial ADU** — limited to state-exemption-compliant JADU/ADU per
  IB-400; ~30-min intake approval.
- **OTC Trade & minor permits** — re-roof, water heater, service change.
- **Self-Certification** for licensed design professionals on residential
  additions/remodels (limited scope — see DSD bulletin).

## Required submittal exhibits — new residential (SFD or ADU)

| Exhibit | Requirement | Reference |
|---------|-------------|-----------|
| Site/Plot Plan | North arrow, scale ≤ 1" = 20', all structures, setbacks, easements, topo where slope > 10%, BMP envelope | **IB-122** |
| Architectural (floor, elev, sec, roof) | Dimensioned, material callouts, window schedule, T24 envelope callouts | — |
| Structural plans + calcs | Foundation, framing, shear schedule, gravity & lateral calcs; ADU may use prescriptive IRC | CBC Ch 16 + local |
| Title 24 Energy — CF1R | HERS-registered; all-electric where reach code applies | CEC 2022/2025 |
| CALGreen residential checklist | Mandatory + local Tier 1 if applicable | **IB-501** and `waste-and-calgreen.md` |
| WMF (Waste Management Form) | SeamlessDocs submission; 65% diversion; refundable deposit | `waste-and-calgreen.md` |
| Landscape + MWELO (> 500 sf landscape) | Hydrozone, irrigation schedule, MAWA/ETWU calcs | SDMC Ch 14 Art 2 Div 4 |
| Soils / Geotechnical report | Required for: slopes > 5:1 on lot, expansive soils, fills, ≥ 2-story w/ variable bearing | **IB-515**; single-fam default bearing **IB-141** |
| Stormwater intake form + WPCP or SWPPP + SWQMP if PDP | Based on threshold (see `stormwater-and-grading.md`) | Storm Water Std Manual |
| Truss calcs / shop drawings | Deferred if signed structural defers | — |
| Fire sprinkler plans | NFPA 13D (SFR) or 13R (R-2) — when required, deferred to SDFD | CFC Ch 9 |
| School Mitigation certification | From SDUSD or applicable district | Ed Code 17620 |
| Water & sewer availability | Public Utilities will-serve; sewer study if > threshold | **IB-122/126** |

## Review disciplines

| Discipline | Reviews |
|-----------|---------|
| Building | CBC/CRC structural + life-safety |

| Stormwater (Grading/SWQ) | WPCP/SWPPP/SWQMP, BMPs |

| Historical | 45+ yr structures |
| Coastal | CDP concurrence |
| Transportation / Engineering | ROW, curb cuts, sidewalks |

## Fees

- **IB-501** — Fee schedule for construction permits/structures.
- **IB-502** — Fee schedule for DSD general services.
- Initial plan-check deposit varies by valuation; reconciled at issuance.
- **Impact fees**: Development Impact Fees (DIF) by community + Facilities
  Benefits Assessment (FBA) in some areas + Transportation Impact Fee.
- **School fees**: Level 1 state cap — verify current rate with OPSC annual update
  (opsc.ca.gov) before citing on submittals; rate changes each fiscal year.
- **WMF deposit**: sliding scale by valuation; refundable per SDMC 66.0702.

## Correction-matrix / resubmittal process

- The cycle issues report lists each comment with a **CIP# / comment #**,
  discipline, and code cite.
- Applicant response = **written narrative** addressing every comment in
  order + **bubbled/cloud'd revised plans** + **red-lined markup** where
  requested.
- Upload sequence in Accela: RESPONSE (PDF narrative) → PLANS (revised PDF,
  single bookmarked file) → SUPPLEMENTS.
- This repo's `plan_check_agent.py` ingests the letter + drawings and
  auto-builds the correction matrix + cross-references to sheet locations.

## Construction Change (CC)

- Filed under the same PRJ once PMT is issued.
- "Minor" vs "Major" CC — major = new square footage, structural re-do,
  use change; minor = field fixes.
- One-cycle review typical if well documented.

## Deferred submittals

- Trusses, solar PV, fire sprinklers, fire alarm, storefront glazing,
  anchor calcs, shop drawings.
- Must be **listed** on approved plan sheet ("Deferred Submittal Items"
  table) to be accepted later without CC.
