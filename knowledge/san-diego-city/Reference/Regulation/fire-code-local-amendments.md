# Fire Code — Local Amendments, SDFD — City of San Diego

Authority: **San Diego Fire-Rescue Department (SDFD) Community Risk Reduction
Division** (formerly Fire Prevention). Adopts California Fire Code (CFC) with
local amendments codified in **SDMC Chapter 5 Article 11 (Fire Code)** and
wildland-urban interface (CWUIC) in **SDMC Ch 5 Article 12**.

**Code cycle flag:** 2025 CFC + California Wildland-Urban Interface Code
(CWUIC) effective **January 1, 2026**. SDFD published proposed local
amendments in Oct 2025; applicants submitting after 1/1/2026 use 2025 code.

## Fire apparatus access roads (SDMC §96.1.503 / CFC §503 + Appendix D)

| Parameter | City standard |
|-----------|---------------|
| **Minimum width — commercial / MF / general** | **24 ft** unobstructed improved width |
| **Minimum width — SFR driveway serving ≤ 2 DU** | **16 ft** unobstructed improved width (with FD-approved gate geometry) |
| **Vertical clearance** | **13 ft 6 in** |
| **Inside turning radius** | **28 ft** minimum |
| **Outside turning radius** | **48 ft** (cul-de-sac or 90° turn) |
| **Cul-de-sac radius** | **36 ft paved / 40 ft graded** min |
| **Grade** | **≤ 15%** (10% preferred; > 10% triggers noncombustible surfacing) |
| **Surface** | All-weather; must support ≥ 75,000 lb imposed loads |
| **Dead-end length** | Approved turnaround required if > **150 ft** |
| **Bridge/vehicle load** | 75,000 lb gross vehicle weight |

Hammerhead / T / Y turnarounds must show fire-apparatus template on plans.

## Hydrants & fire flow (CFC Appendix B/C + local)

- **R-3 detached (SFR, duplex, ADU)**: 1,500 gpm @ 20 psi residual minimum;
  ≥ 1 hydrant within 600 ft of farthest exterior wall.
- **R-2 / Commercial**: fire flow per CFC Table B105.1 (1,500–8,000 gpm)
  based on construction type and area; hydrant spacing per C105.1
  (250–400 ft).
- Sprinklered buildings receive reductions per CFC B105.2 (up to 75%).
- Ask **Public Utilities** for flow test letter; dated ≤ 1 year.

## Defensible space / brush management

SDMC §142.0412 — applies to parcels adjacent to or within native/undisturbed
vegetation areas (mapped **VHFHSZ** or CAL FIRE Local Responsibility Area):

| Zone | Extent | Requirements |
|------|--------|--------------|
| **Zone 1** | 0–35 ft from structure | Irrigated, well-maintained; remove dead fuel; ornamental plantings limited; tree canopy spacing ≥ 3× height separation; no woody shrub within 5 ft of structure |
| **Zone 2** | 35–100 ft (or to lot line) | Thinned native vegetation; 50% fuel reduction; vertical separation (no ladder fuels); crown-to-crown spacing by slope |

Deeded BMZ easements may extend onto adjacent parcels; show on site plan.

## Wildland-Urban Interface (WUI) / Chapter 7A CBC

- **Very High Fire Hazard Severity Zone (VHFHSZ)** mapping by CAL FIRE /
  SDFD. Within VHFHSZ + "WUI Fire Area" triggers:
  - **CBC Chapter 7A** exterior materials (ignition-resistant exterior,
    Class A roof, ≤ 1/8" mesh vent screens, tempered/dual-pane glazing,
    noncombustible decks within 10 ft, etc.).
  - **CWUIC** additional requirements as of 2026 code cycle.
- Applies to new construction, additions ≥ 50% of original value, and
  re-roofs where ≥ 50% replaced.

## Sprinkler thresholds

| Occupancy | Sprinkler requirement |
|-----------|------------------------|
| **R-3 SFR / duplex / townhome** | NFPA 13D required for all **new** SFR (CRC R313) — has been state law since 2011 |
| **Detached ADU** | **Not required** if (a) primary dwelling not sprinklered AND (b) ADU ≤ 1,200 sf — per state ADU law and IB-400. Verify BMZ/VHFHSZ exceptions. |
| **R-2 multifamily** | NFPA 13R up to 4 stories / ≤ 60 ft; NFPA 13 above |
| **Commercial** | CBC 903.2 thresholds (often 5,000–12,000 sf depending on use) |
| **VHFHSZ** | Any new structure ≥ 500 sf — 13D regardless of primary |

## Fire alarm thresholds

- R-2 ≥ 16 DU: manual alarm system (CFC 907.2.9).
- Single-station smoke/CO alarms in all SFR/ADU bedrooms + hallways + every
  floor (CRC R314/R315).

## Address numbering / building identification (CFC §505)

- **≥ 4 in tall** numerals, contrasting color, visible from access road.
- ≥ 6 in for multi-unit; ≥ 12 in for commercial with long setback.
- Suite/unit ID at each tenant entrance + building directory.
- ADU must have **distinct address** or **unit designator** per Postmaster +
  City addressing.

## Knox Box / access

- Knox Box required at all buildings with alarm panels, gated commercial
  properties, and multi-family with controlled access.
- Knox key switches on vehicle gates over fire access.
