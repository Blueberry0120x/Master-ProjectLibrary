# Garden Grove — ADU & SB 9

Source: California Gov Code §65852.2 (ADU law) + Garden Grove Municipal Code Title 10.
State ADU law overrides any conflicting local provisions — when in doubt, state law governs.

---

## 1. State ADU Law Floor (Gov Code 65852.2)

All California jurisdictions must comply with the state ADU law as a minimum.
See `../_shared/statewide-adu-law.md` for full state-law coverage.

Key state minimums:
- **Ministerial approval** — no discretionary review, no CEQA, no public hearing.
- **By-right on any single-family or multifamily lot** in residential or mixed-use zone.
- **Junior ADU (JADU):** up to 500 sf, within existing primary structure, one per lot.
- **Detached ADU:** up to 1,200 sf (or 50% of primary dwelling sf if larger).
- **Attached ADU:** up to 50% of primary dwelling area, max 1,200 sf.
- **Setbacks:** 4 ft rear and side for new detached ADU; 0 ft for conversions of existing structures.
- **Owner-occupancy requirement:** suspended statewide through Jan 1, 2025. Verify current status.
- **Parking:** Not required within ½ mile of transit or for conversions of existing structures.
- **No minimum lot size** for ADU.
- **Fees:** Cannot exceed proportional share of actual cost.

## 2. Garden Grove Local ADU Standards

Garden Grove's ADU ordinance must comply with state law. Where local standards are
more restrictive than state law, state law prevails.

### Permitted Locations

- Any RS zone (RS-1 through RS-4) — one ADU + one JADU per lot with existing or proposed SFD.
- RM zones — state law allows ADUs; check local ordinance for additional allowances.
- Mixed-use zones — permitted by state law where residential is permitted.

### ADU Size Limits

| Type | Maximum Size |
|------|-------------|
| JADU | 500 sf (interior conversion of existing structure) |
| Attached ADU | Lesser of 50% of main dwelling sf or 1,200 sf |
| Detached ADU | 1,200 sf (or 50% of primary sf per state floor) |
| Conversion ADU | No size increase beyond existing footprint (unless attached addition) |

### Setbacks

| Type | Setback |
|------|---------|
| New detached ADU — rear | 4 ft minimum (state law) |
| New detached ADU — side | 4 ft minimum (state law) |
| New detached ADU — front | Per base zone front setback |
| Conversion of existing structure | May be 0 ft if existing nonconforming setback |

### Height

- Detached ADU: Maximum 16 ft when within 4 ft of rear/side property line.
- Above a garage (ADU over garage): Up to 24 ft if base zone allows.
- Attached ADU: Limited by base zone height.

### Parking

- Not required for ADUs within ½ mile of public transit.
- Not required for conversion of existing space (garage, storage, etc.).
- If required: 1 space per ADU unit (tandem in existing driveway permitted).

### Design Standards

Garden Grove may apply objective design standards per AB 2221 (effective Jan 1, 2023).
These may include:
- Exterior materials consistent with primary dwelling.
- Windows and doors facing street must be architecturally compatible.
- Separate entrance for ADU (not through primary dwelling interior).

**Verify current objective standards with Planning Division — (714) 741-5140.**

## 3. Permit Process for ADU

1. Submit to Building Division — ministerial, no discretionary review.
2. Building Division reviews for objective standards + state ADU law compliance.
3. **65-day approval deadline** from complete application — state law mandate.
4. If denied, city must provide written findings; denial is rare for ministerial ADU.
5. Permit fees cannot be imposed for JADU or ADU under 750 sf that is within ½ mile of transit.

---

## 4. Bonus ADUs (Multifamily)

On multifamily lots (RM zones):
- Up to 2 detached ADUs per lot (by-right, ministerial).
- Conversion ADUs = 25% of existing multifamily units (rounded down).

Example: 10-unit apartment building → up to 2 conversion ADUs + 2 detached ADUs.

## 5. SB 9 — Two-Unit / Urban Lot Split

Gov Code §§65852.21 and 66411.7 (effective Jan 1, 2022).

### Two-Unit Development (65852.21)

- Allows 2 primary units on any single-family lot in an urbanized area.
- Both units can have an ADU + JADU → up to 4 units total possible.
- Ministerial approval; no discretionary review.
- Setbacks: 4 ft rear and side (same as ADU).
- Not permitted on: historic landmark parcels, floodplains, farmland, fire hazard zones
  (verify Garden Grove WUI designation), wetlands, slopes > 25%.
- Owner-occupancy required for 3 years for lot split (not for 2-unit development).

### Urban Lot Split (66411.7)

- Splits existing SFD parcel into two lots of approximately equal size.
- Minimum lot size post-split: 1,200 sf each.
- Each new lot can have 2 units + ADU + JADU.
- Tentative Parcel Map required; ministerial approval.
- Not permitted on same exclusions as two-unit development.

**Note:** Garden Grove must grant SB 9 ministerially. Verify that parcel is not in an
excluded category before proceeding.

## 6. Common Issues in Garden Grove

- **Fire sprinklers.** GGFD typically requires NFPA 13D fire sprinklers for new detached ADUs
  if the primary dwelling is required to have sprinklers. Confirm with GGFD at (714) 741-5550.
- **School fees.** Level 1 school fees apply to ADUs > 500 sf. Payable to Garden Grove
  Unified School District (GGUSD) before permit issuance.
- **CDDMP.** Construction & Demolition Debris Management Plan required for any demolition
  or new construction. See `calgreen-energy.md`.
- **Utility connections.** New water/sewer connections for ADU require Public Works review.
  Capacity fees may apply.
