# City of Garden Grove — Plan-Check Reference Library

Master jurisdictional reference for City of Garden Grove permit submittals.
Scope: Single-family (SFD), ADU/JADU, additions, accessory structures, and small commercial.
Located in Orange County, CA.

Code cycle baseline: **2022 California codes** (CBC/CRC/CPC/CMC/CEC/CALGreen/CFC)
as locally amended per Garden Grove Municipal Code (GGMC).
**2025 cycle effective Jan 1, 2026.**

## Global Notes / Gotchas

- **Orange County context.** Garden Grove is a city within OC. For unincorporated parcels,
  use `orange-county/` instead. City parcels are fully within Garden Grove jurisdiction.
- **Permit portal.** Garden Grove may use a hosted Accela or MyGov portal — verify the
  current URL at the Building Division counter before submitting online.
- **Fire sprinkler threshold.** GGFD requires NFPA 13D residential fire sprinklers for
  new SFD construction per local amendment to CRC R313. Confirm ADU trigger.
- **2025 code cycle.** Verify with Building Division which cycle applies to your submittal
  date — Garden Grove typically adopts within 30–60 days of California effective date.
- **CALGreen WMF equivalent.** Garden Grove uses a Construction & Demolition Debris
  Management Plan (CDDMP) instead of San Diego's WMF. See `calgreen-energy.md`.
