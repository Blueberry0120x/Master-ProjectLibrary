# Garden Grove — Forms & Submittals Index

All forms available from the Community Development Dept — Building Division
unless noted. (714) 741-5165 | https://www.gardengrove.org/building

---

## Permit Application Forms

| Form | Name | Notes |
|------|------|-------|
| Building Permit Application | Standard permit application | Submit at counter or via online portal |
| Owner-Builder Declaration | Required when owner acts as GC | No licensed contractor |
| Contractor/Subcontractor Disclosure | Lists all trade contractors | Submit with application |
| Soils Report Waiver (if applicable) | Request to waive soils report | Small additions on flat lots |

## CALGreen / C&D Forms

| Form | Name | When Required |
|------|------|--------------|
| Construction & Demolition Debris Management Plan (CDDMP) | C&D waste diversion plan | All new construction + demolition |
| CDDMP Completion Form | Post-construction diversion documentation | Submit with hauler receipts at project close |
| CALGreen Residential Checklist | Mandatory measures checklist (Part 11 Ch 4) | Attach to plan check submittal |
| CALGreen Non-Residential Checklist | Mandatory measures checklist (Part 11 Ch 5) | Non-residential projects |

## Notes

- **No WMF.** Garden Grove uses a CDDMP, not San Diego's SeamlessDocs WMF.
- **Diversion target:** 65% by weight per CALGreen 4.408.
- **CDDMP must be submitted** with the building permit application and a completion
  form (with hauler receipts) must be filed at project close.
