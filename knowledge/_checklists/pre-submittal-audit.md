# Pre-Submittal Audit Checklist

Use this checklist before any first-cycle plan check submittal.
Completing this audit before submittal typically saves 1–2 correction cycles.

---

## 1. Project Identity

- [ ] Project address confirmed and correct on all sheets and forms
- [ ] APN verified (cross-check county assessor records)
- [ ] Zoning confirmed (verify on jurisdiction's GIS map, not Google Maps)
- [ ] Applicable overlay zones identified (coastal, WUI, TPA, historic, flood)
- [ ] PRJ/permit number included on all documents (if pre-assigned)
- [ ] Applicant / owner name matches permit application exactly

---

## 2. Zoning & Site Compliance

- [ ] Zone confirmed; correct development standards table used
- [ ] All setbacks dimensioned on site plan (front, both sides, rear)
- [ ] FAR calculated (if applicable) and shown on cover sheet
- [ ] Lot coverage calculated and shown; within zone maximum
- [ ] Building height calculated from lower of existing/finished grade to highest element
- [ ] Height within zone maximum; architectural projection exceptions cited if used
- [ ] Parking count shown; meets zone requirements
- [ ] Applicable overlays addressed (coastal height, WUI materials, historic, TPA)

---

## 3. Site Plan Completeness

- [ ] North arrow present
- [ ] Scale ≤ 1 in = 20 ft (or as required by jurisdiction)
- [ ] All property lines with dimensions
- [ ] All existing and proposed structures shown
- [ ] Setbacks dimensioned to nearest property line
- [ ] Easements (public, private, utility) shown and labeled
- [ ] Street name(s) labeled
- [ ] Topography shown if slope > 10% or required by jurisdiction
- [ ] Drainage flow arrows
- [ ] Utility locations (water, sewer, gas, electric)
- [ ] BMP/stormwater features shown (where required)
- [ ] Trees: existing significant trees identified

---

## 4. Architectural Drawings

- [ ] All floor plans dimensioned; rooms labeled
- [ ] All elevations (four faces) with materials called out
- [ ] Sections: minimum one building section through structure
- [ ] Roof plan with drainage, slopes, ridge/valley locations
- [ ] Door and window schedule with sizes, types, glazing specs
- [ ] Accessibility: if applicable (threshold, etc.)

---

## 5. Structural

- [ ] Foundation plan: slab, footing, or raised floor — fully detailed
- [ ] Framing plan: floor and roof framing members sized and spaced
- [ ] Shear wall schedule: locations, nailing, hold-down hardware
- [ ] Gravity and lateral load calculations (if engineered)
- [ ] Soils report referenced on foundation plan (or waiver obtained)
- [ ] Geotechnical: bearing pressure on foundation notes
- [ ] Truss: deferred submittal listed on plans if manufactured trusses used

---

## 6. Energy — Title 24 Part 6

- [ ] CF1R generated for correct climate zone (see `calgreen-energy.md`)
- [ ] CF1R attached to plan check set
- [ ] All mandatory measures noted on plans (insulation, duct sealing, WH, EV conduit)
- [ ] Solar-ready zone shown on roof plan
- [ ] EV conduit noted on electrical plan
- [ ] HERS-required measures listed on CF1R; HERS registration plan confirmed

---

## 7. CALGreen

- [ ] CALGreen residential checklist complete and attached
- [ ] Low-flow fixture schedule on plans matches §4.303.1 limits
- [ ] Bathroom exhaust fans specified with CFM rating (§4.506.1)
- [ ] HVAC sizing: Manual J per ACCA noted (§4.507.2)
- [ ] VOC limit compliance noted for adhesives, sealants, paints (§4.504.1)
- [ ] Vapor retarder under slab called out on foundation detail (§4.505.1)

---

## 8. Construction & Demolition Waste

- [ ] C&D Waste Management Plan / WMF submitted (per jurisdiction)
- [ ] Total estimated tonnage shown (use debris factor × SF)
- [ ] Diversion percentage ≥ 65%
- [ ] Recycling and disposal facility names and addresses listed
- [ ] City of San Diego: WMF via SeamlessDocs submitted separately

---

## 9. Stormwater

- [ ] Project tier determined (Standard vs. Priority Development Project)
- [ ] BMP site plan submitted showing all BMPs
- [ ] If PDP: SWQMP / WQMP prepared by licensed civil engineer
- [ ] Erosion control plan included if grading permit triggered
- [ ] Construction SWPPP ready if land disturbance ≥ 1 acre
- [ ] Flood zone status confirmed (FEMA MSC check); BFE compliance shown if in Zone AE

---

## 10. Fire

- [ ] Smoke alarm locations shown on floor plan (CRC R314)
- [ ] CO detector locations shown (CRC R315)
- [ ] Fire sprinkler note on plans (deferred or designed)
- [ ] Fire access and addressing: street numbers on elevations
- [ ] WUI parcels: Chapter 7A materials specified on elevations and material schedule
- [ ] Defensible space zones shown on site plan if VHFHSZ parcel
- [ ] Hydrant location verified within required distance (check with AHJ)

---

## 11. School Fees

- [ ] Applicable school district(s) identified
- [ ] Level 1 fee certification(s) obtained or scheduled
- [ ] Fee amount budgeted (paid before permit issuance)

---

## 12. Utility Coordination

- [ ] Water / sewer will-serve letter obtained (or scheduled)
- [ ] Capacity fees estimated and budgeted
- [ ] Utility trenching / connection shown on site plan

---

## 13. ADU-Specific (if applicable)

- [ ] ADU type confirmed: JADU, attached, detached, conversion
- [ ] State ADU law compliance verified (setbacks, size, height, ministerial)
- [ ] IB-400 (San Diego) or equivalent jurisdiction bulletin reviewed
- [ ] Separate entrance shown on floor plan
- [ ] Separate address or unit number assigned
- [ ] Parking: state exemption confirmed if within ½ mile of transit
- [ ] Fire sprinkler trigger confirmed with AHJ

---
