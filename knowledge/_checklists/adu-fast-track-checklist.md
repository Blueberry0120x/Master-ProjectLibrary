# ADU Fast-Track Readiness Checklist

Use before submitting any ADU or JADU permit application.
State law mandates ministerial approval within 65 days of a complete application.
A complete, compliant first submittal triggers the clock; a deficient one resets it.

---

## 1. Confirm ADU Type

- [ ] **JADU** (Junior ADU): ≤ 500 sf, within existing primary structure, one per SFD lot
- [ ] **Attached ADU**: addition to primary dwelling; ≤ 50% of primary, max 1,200 sf
- [ ] **Detached ADU**: separate structure on same lot; max 1,200 sf
- [ ] **Conversion ADU**: garage, storage, or other space converted to living unit
- [ ] **Above-garage ADU**: ADU constructed over existing or new garage

---

## 2. Site & Zoning

- [ ] Lot is in a residential or mixed-use zone (RS, RM, RX, R-1, SF, etc.)
- [ ] Primary SFD or multifamily structure exists or is proposed on the lot
- [ ] No more than one ADU + one JADU per SFD lot (or bonus ADU rules for MF)
- [ ] Parcel is not in an excluded category:
  - [ ] Not a historic landmark (individually designated)
  - [ ] Not in a floodway
  - [ ] Not on prime farmland
  - [ ] Not on slopes > 25% (SB 9 only; standard ADU may still be allowed)

## 6. Parking

- [ ] Parcel within ½ mile of public transit stop → **0 parking required**
- [ ] Conversion of existing garage/covered space → **0 replacement required**
- [ ] ADU in historic district → **0 parking required** (state law)
- [ ] ADU in TPA → **0 parking required**
- [ ] Otherwise: 1 space per ADU unit; tandem/driveway counted

---

## 7. Required Submittal Documents

- [ ] Site plan showing ADU location, setbacks, all structures
- [ ] Floor plans (ADU + existing primary)
- [ ] Elevations (all four faces of ADU)
- [ ] Sections
- [ ] Foundation plan
- [ ] Title 24 CF1R (separate from primary if standalone system)
- [ ] CALGreen checklist
- [ ] C&D Waste Management Plan / WMF (per jurisdiction)
- [ ] Fire sprinkler note or deferred submittal designation (verify with AHJ)
- [ ] School fee certification (if ADU > 500 sf)
- [ ] Separate entrance shown on floor plan

---

## 8. Fees

- [ ] ADU ≤ 750 sf within ½ mile of transit: no impact fees (state law)
- [ ] ADU connected to existing systems: proportional connection fees only
- [ ] School fees: Level 1 if ADU > 500 sf (varies by jurisdiction)
- [ ] Plan check fee: reduced for ADUs in many jurisdictions — confirm at counter

---

## 9. Fire Sprinklers

### Step 1 — Is the primary dwelling sprinklered?

- [ ] **Primary built after 2011** → likely has NFPA 13D (CRC R313 mandatory since 2011)
  → **ADU sprinklers REQUIRED** — tie into primary system or install standalone 13D
- [ ] **Primary built before 2011** → likely unsprinklered
  → **ADU sprinklers WAIVED** under Gov Code §65852.2 (state preemption)

### Step 2 — Override checks (even if primary is unsprinklered)

- [ ] Is the site in **VHFHSZ** (Very High Fire Hazard Severity Zone)?
  → Confirm with local fire dept — may be required regardless
- [ ] Is this a **Bonus ADU** (San Diego deed-restricted affordable program)?
  → Sprinklers always required
- [ ] Is this a **Density Bonus Affordable ADU**?
  → Sprinklers always required

### Step 3 — Construction type check (common misconception)

> **Type V-A (1-hr rated) does NOT waive sprinklers.**
> The 1-hr fire-resistance rating is a structural framing requirement.
> CRC R313 sprinklers are triggered by dwelling type + primary sprinkler status — completely independent of construction type. A V-A ADU still needs sprinklers if the primary is sprinklered.

## 10. 65-Day Clock

- Application is complete when ALL required items are submitted.
- Jurisdiction must approve or deny within 65 days of complete application.
- If denied: written findings required; denial of ministerial ADU is rare.
- If approved: permit ready for issuance after fee payment.

---

## 11. Post-Permit

- [ ] Separate address or unit number assigned to ADU
- [ ] Certificate of Occupancy issued for ADU (separate from primary)
- [ ] Utility metering: confirm shared vs. separate meter with utility provider
- [ ] Rental disclosure: if renting ADU, comply with local rental ordinance
