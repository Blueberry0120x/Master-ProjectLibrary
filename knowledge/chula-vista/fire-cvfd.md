# Chula Vista Fire Department (CVFD) — Code & Standards

CVFD enforces the **California Fire Code (CFC) 2022 edition** with local amendments
adopted in **CVMC Title 15** (Buildings and Construction). Next cycle (2025 CFC)
effective **Jan 1, 2026**.

CVFD Fire Prevention Bureau reviews all building permits for fire code compliance and
provides approved access, hydrants, sprinklers, alarm, and hazardous-materials review.

## Fire Apparatus Access Roads (CFC Ch 5)

| Standard | Value |
|---|---|
| Minimum unobstructed width | **20 ft** (road); **26 ft** adjacent to hydrant |
| Vertical clearance | 13 ft 6 in |
| Turning radius (inside) | 28 ft (verify — CVFD may require 32 ft inside / 52 ft outside) |
| Dead-end limit | **150 ft** max without turnaround |
| Turnaround | 120-ft hammerhead or Ø96-ft cul-de-sac |
| Grade | 15% max (some CVFD approvals up to 20% with mitigation) |
| Surface | All-weather, 75,000-lb fire apparatus load |
| Access required for any portion of facility | Within 150 ft of ALL exterior walls (ground level) |

**Driveway/access** to single-family in hillside / VHFHSZ areas: 16 ft min clear width
(CVFD may require 20 ft), with turnouts every 400 ft if >150 ft long.

## Fire Hydrants (CFC 507, Appendix C)

| Standard | Value |
|---|---|
| Max spacing (residential) | 500 ft |
| Max spacing (commercial) | 300–400 ft |
| Max distance from structure | **400 ft** (SFD w/ sprinklers), 600 ft sometimes allowed |
| Min flow (SFD, NFPA 13D) | **1,000 gpm @ 20 psi for 1 hr** (reducible) |
| Min flow (commercial / multifamily) | Per CFC Appendix B (1,500–8,000 gpm based on type/area) |
| Sprinkler reduction | Up to 75% (never below 1,500 gpm commercial, 1,000 gpm SFD) |
| Hydrant type | AWWA C502, 2×2½" + 1×4" or 6" steamer (verify per water district) |

**Flow test** required for new development — coordinate with Otay Water or Sweetwater.

## Fire Sprinklers

### Residential (NFPA 13D per CRC R313)

- **All new one- and two-family dwellings** — sprinklers required per CRC R313.2
- **All new ADUs** — sprinklers required if primary dwelling has sprinklers (R313.2 exception)
- Design: NFPA 13D, typical 4-head calc, attic heat detectors where required

### Commercial / Multifamily (NFPA 13 / 13R)

- **R-1/R-2 multifamily** — required if 3+ stories or >3 units, or per CBC 903.2.8
- **A, B, E, I, M, S** — per CBC 903.2 (occupancy-specific thresholds)
- Commercial triggers: >12,000 sf (typical), high-pile storage, assembly >300 occupants
- **FDC** location approved by CVFD; typically within 100 ft of hydrant

### Alarm / Monitoring

- Sprinkler monitoring per NFPA 72 (central station UL-listed)
- Manual pull + voice evac per occupancy

## Very High Fire Hazard Severity Zone (VHFHSZ) / WUI

Per **CAL FIRE FHSZ maps** (updated 2024, effective 2025), Chula Vista has VHFHSZ in:
- **Eastern territories** — Otay Ranch east, Millenia, Bonita-adjacent areas, canyon-rim
  properties along Otay River, Rice Canyon, Salt Creek
- Areas west of Wildland-Urban Interface along canyons

**Verify parcel-specific designation** at: https://osfm.fire.ca.gov/what-we-do/community-wildfire-preparedness-and-mitigation/fire-hazard-severity-zones

### Applicable Requirements in VHFHSZ

| Requirement | Source |
|---|---|
| **CBC Chapter 7A** — WUI materials | 2022 CBC |
| Ignition-resistant exterior | Ch 7A |
| Class A roof covering | Ch 7A / CRC R327 |
| Boxed / fire-resistant eaves, soffits, vents | §706A, §706A.3 |
| Ember-resistant vents | WUI-listed (BML) |
| Dual-pane tempered windows | §708A |
| Noncombustible 0–5 ft zone (Zone 0) | PRC 4291 / future |
| 100 ft defensible space | PRC 4291 / Govt Code 51182 |
| Decks noncombustible | §709A |

### Defensible Space (PRC 4291)

- **Zone 0** (0–5 ft): ember-resistant, non-combustible materials, no plants/mulch (state rule pending rulemaking)
- **Zone 1** (5–30 ft): lean, clean, green — low plants, 3× height between tree canopies
- **Zone 2** (30–100 ft): reduced fuel, 10-ft vertical separation from ground to canopy

Annual inspections in VHFHSZ by CVFD brush management inspectors.

## Hazardous Materials

- CUPA administered by **San Diego County Department of Environmental Health** (DEH)
- CVFD reviews HMMP / HMBP for inventory thresholds
- CFC Ch 50-67 hazmat provisions
- Above-ground fuel tanks — CVFD + DEH

## Key Local Amendments (typical CVMC 15.36)

- Sprinkler required for any new SFD (aligns with CRC R313, no local waiver)
- Fire lane signage / red curb requirements
- Knox Box required on all non-residential and gated multifamily
- Address visibility: 4-in min (6-in for commercial), contrasting, illuminated at night
- False alarm fees per Title 5
