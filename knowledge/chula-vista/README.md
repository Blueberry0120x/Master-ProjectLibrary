# Chula Vista, CA — Development & Building Reference Library

Master reference for development, building, fire, and environmental regulations in the
**City of Chula Vista** (South Bay, San Diego County). Population ~280,000; second-largest
city in San Diego County.

## Regulating Authorities

| Authority | Scope |
|---|---|
| **Development Services Department (DSD)** | Building, Land Development, Planning, Landscape |
| **Chula Vista Fire Department (CVFD)** | CFC enforcement, fire prevention, access, sprinklers, WUI |
| **Engineering Department** | Grading, stormwater, right-of-way, traffic, sewer |
| **Public Works** | ROW permits, streets, sidewalks |
| **Otay Water District** | Water/recycled water east of I-805 (eastern Chula Vista) |
| **Sweetwater Authority** | Water service west of I-805 (western Chula Vista) |
| **San Diego County DEH** | Food, pools, hazmat CUPA |
| **San Diego Regional Water Quality Control Board** | MS4 NPDES permit (Region 9) |

## Code Framework

Chula Vista Municipal Code (CVMC) — authority per Charter. Key titles:

| Title | Subject |
|---|---|
| **Title 15** | Buildings and Construction (adopts CBC, CRC, CFC, CMC, CPC, CEC, CALGreen, Energy) |
| **Title 17** | Environmental Quality |
| **Title 18** | Subdivisions |
| **Title 19** | Planning and Zoning |
| Title 14 | Grading |
| Title 8.25 | Construction & Demolition Debris Recycling |

### California Code Cycle

- **2022 California Building Standards Code** currently effective (since Jan 1, 2023)
- **2025 California Building Standards Code** effective **Jan 1, 2026**
- Chula Vista adopts via Title 15 with local amendments (climatic, geologic, topographic findings)

## Jurisdictional Notes

- **Bayfront** — governed by Chula Vista Bayfront Master Plan + Port of San Diego jurisdiction for tidelands
- **Eastern territories** — large Planned Communities (Otay Ranch, EastLake, Rancho del Rey, Sunbow) with their own SPAs / PC district regulations that supersede base zoning
- **Urban Core Specific Plan (UCSP)** — downtown Third Avenue, mixed-use form-based
