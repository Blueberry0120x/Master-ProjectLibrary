# Chula Vista — Permits & Submittals

Building permits in Chula Vista are processed through the **Development Services Department
(DSD)** via the **Accela Citizen Access** portal. DSD integrates Building, Planning,
Land Development/Engineering, and coordinates with CVFD.

## Accela Citizen Access

**Portal:** https://aca-prod.accela.com/CHULAVISTA/

- Applicant registers as a licensed contractor or owner-builder
- Upload PDFs (drawings, calcs, reports) as electronic submittal
- Resubmittals via the same record ID (corrections cycle)
- Fees paid online
- Inspection scheduling through Accela

## Required Submittals — New Residential (SFD / ADU)

| Item | Required |
|---|---|
| Building Permit Application | Yes |
| Site / Plot Plan (setbacks, topo, drainage) | Yes |
| Architectural drawings (floor plans, elevations, sections) | Yes |
| Structural drawings + calcs (stamped) | Yes |
| Title 24 Energy Compliance (CF1R) | Yes |
| CALGreen checklist | Yes |
| Soils / geotechnical report | Yes (new construction) |
| Truss calcs (deferred OK) | If prefabricated |
| MEP plans | Yes |
| Grading / drainage (if ≥50 cy or PDP threshold) | Conditional |
| SWPPP / WPCP | Conditional (construction SW) |
| School fee certificate | Yes |
| Water/sewer availability letters | Yes |
| C&D Debris Management Plan | Yes (see calgreen-energy.md) |
| Fire sprinkler plans (NFPA 13D) | Yes (all new SFD per CRC R313) |

## Plan Review Disciplines

| Reviewer | Scope |
|---|---|
| **Building** | CBC/CRC, structural, accessibility, CALGreen, Title 24 |
| **Planning / Zoning** | Setbacks, FAR, coverage, design, landscape, tree preservation |
| **Land Development / Engineering** | Grading, drainage, SWMM BMPs, ROW, utilities |
| **CVFD** | Access, hydrants, sprinklers, WUI, defensible space |
| **Landscape** | MWELO, Landscape Manual compliance (multifamily / commercial) |
| **Otay Water / Sweetwater** | Water service availability + meter sizing |
| **Sewer (City)** | Capacity / lateral connection |
| **Traffic Engineering** | Driveway, sight, TIA trigger |
| **Environmental** | CEQA (entitlements only) |

## Entitlement Types (pre-building permit)

| Entitlement | Trigger | Authority |
|---|---|---|
| **Design Review** | Multifamily (4+ units), commercial, industrial | DRC / Zoning Admin |
| **Conditional Use Permit (CUP)** | Listed conditional uses per zone | Zoning Admin or Planning Commission |
| **Variance** | Deviation from development standard | Zoning Admin / Planning Commission |
| **Tentative Map (Subdivision)** | 5+ parcels | Planning Commission / Council |
| **Tentative Parcel Map** | 4 or fewer parcels | Zoning Admin |
| **Specific Plan Amendment** | PC / UCSP / Bayfront mods | Council |
| **Rezone** | Zoning change | Council |
| **Precise Plan** | Required on "P" overlay lots | DRC / Zoning Admin |

**Note:** SFD and most ADU permits are **by-right** — no entitlement required unless a
deviation is sought.

## Typical Plan Review Timeline

| Phase | Duration (estimate) |
|---|---|
| First submittal review | 4–6 weeks (residential), 6–8 weeks (commercial) |
| Second review (resubmittal) | 3–4 weeks |
| Third + subsequent | 2–3 weeks |
| ADU (state-mandated) | 60 days max per Gov Code 65852.2 |
| SB 9 lot split + unit | 60 days ministerial |
| Over-the-counter / minor | same day – 1 week (T.I.s, minor additions) |

## Plan Check Fees (structure)

- Based on valuation (ICC Building Valuation Data) or fixed schedule
- **Building plan check** ≈ 65% of building permit fee (paid at submittal)
- **Planning plan check** fixed fees by permit type
- **Engineering / Grading** per cubic yard or valuation
- **Fire review** per square footage schedule
- **SMIP** (Strong Motion Instrumentation) — state fee $0.00013/$1 valuation residential
- **Green Building fee** per state schedule
- **DIF** (Development Impact Fees): Traffic, Park, Drainage, Public Facilities, Fire,
  Library, Sewer — paid at **permit issuance** (not submittal)
- **School fees** — CVESD / Sweetwater UHSD / Chula Vista ESD

Fee schedule: https://www.chulavistaca.gov/departments/development-services (current Master Fee Schedule)

## Inspections

- Scheduled via Accela; typically 24-hr advance notice
- Inspections: Foundation/setback, underground plumbing, rough frame, rough MEP, insulation,
  drywall, final
- Special inspections per CBC Ch 17 (geotech, steel, concrete ≥2500 psi, masonry, welding)
- **CVFD** inspections: rough sprinkler, final sprinkler, hydrant flow test, FDC

## Water & Sewer Services

### Otay Water District (east of I-805, approx.)

- **Service area:** Eastern Chula Vista, EastLake, Otay Ranch, Sunbow, Bonita
- Water availability letter, meter sizing, backflow required
- Recycled water required where available (irrigation-only)
- Contact: (619) 670-2222 — https://otaywater.gov

### Sweetwater Authority (west of I-805, approx.)

- **Service area:** Western Chula Vista, National City, Bonita
- Joint powers: South Bay Irrigation District + National City
- Water availability, capacity fees
- Contact: (619) 420-1413 — https://sweetwater.org

### Sewer

- City of Chula Vista sewer system (operates collection)
- Connects to Metropolitan Wastewater Department (City of San Diego) Point Loma / South Bay
- Sewer capacity fee at issuance; lateral inspection required

## Resubmittal Protocol

1. Address each correction in writing (response letter)
2. Revisions clouded + deltas on drawings
3. Resubmit full set via Accela to same record
4. Route to original reviewers
