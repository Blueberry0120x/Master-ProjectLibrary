# County of San Diego (Unincorporated) — Reference Library

**Jurisdiction scope:** Unincorporated areas of San Diego County (~3,572 sq mi, ~490,000 residents). Does **NOT** cover the 18 incorporated cities (City of San Diego, Chula Vista, Escondido, Oceanside, Carlsbad, etc.), each of which has its own building/zoning authority.

> **CRITICAL:** County of San Diego and City of San Diego are distinct jurisdictions with different codes, fees, forms, zoning, and submittal portals. Never confuse "San Diego" (City) with "County of San Diego" (unincorporated). See `../san-diego-city/` for the City.

## Key Codes & Ordinances

| Code | Citation | Notes |
|------|----------|-------|
| **County Zoning Ordinance** | Title 6, SD County Code | Uses designator system (Use/Lot Size/Height/Coverage/Yard/Animal) |
| **Consolidated Building Code** | Title 9, Div 1, SD County Code | Adopts 2022 CBC, CRC, CEC, CMC, CPC, CEnC, CALGreen, CFC with County amendments |
| **Fire Code** | Title 9, Div 6 | Adopts 2022 CFC w/ amendments; WUI applies countywide |
| **Grading Ordinance** | Title 8, Div 7, Ch 2 (§87.201+) | Clearing/grading threshold 200 CY or 2,500 SF |
| **Watershed Protection Ordinance (WPO)** | Title 6, Div 8, Ch 7 (§67.801+) | Replaces former Storm Water Ordinance; adopted 2020 |
| **Resource Protection Ordinance (RPO)** | §86.601+ | Sensitive lands: wetlands, floodplain, steep slopes, sensitive habitat |
| **Biological Mitigation Ordinance / MSCP** | §86.501+ | Multiple Species Conservation Program (South County Subarea Plan) |
| **Light Pollution / Dark Sky** | §59.101+ | Stricter in East County (Palomar/Mount Laguna) |
| **Noise Ordinance** | §36.401+ | Sound level limits by zone |
| **Subdivision Ordinance** | Title 8, Div 1 | Tentative Maps, Parcel Maps |

**2025 code cycle:** CA adopts new triennial codes January 1, 2026. Until then, 2022 CBC/CRC/CFC/CALGreen apply. County intermediate code update typically follows ~90-180 days after state effective date.

## Geographic / Jurisdictional Notes

- Unincorporated County includes communities: **Alpine, Bonsall, Borrego Springs, Boulevard, Campo, Crest, Descanso, Fallbrook, Jamul, Julian, Lakeside, Mt Laguna, Pine Valley, Potrero, Rainbow, Ramona, Rancho Santa Fe, Spring Valley, Valley Center, Warner Springs**, and portions of coastal corridor.
- **28 Community Planning Areas (CPAs)** — each with adopted Community Plan that layers onto Zoning Ord.
- **General Plan (2011 update)** groups lands into Regional Categories: **Village**, **Semi-Rural**, **Rural**.
