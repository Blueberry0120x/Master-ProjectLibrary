# California Codes Overview (Title 24)

**Scope:** Structure of the California Building Standards Code (Title 24 CCR), the 12 Parts, the 2022 vs 2025 adoption cycles, and how to read a California code citation (state model + local amendments).

## 1. What is Title 24?

Title 24 of the California Code of Regulations (CCR) is the **California Building Standards Code**. It is promulgated by the **California Building Standards Commission (CBSC)** under Health & Safety Code Div. 13, Part 2.5. Title 24 adopts model codes (primarily ICC family + NFPA 70) and overlays them with California-specific amendments proposed by state agencies (HCD, OSHPD, DSA-SS, DSA-AC, CEC, SFM, CDPH, BSC) via triennial adoption cycles.

Official home: **dgs.ca.gov/BSC**

## 2. The Twelve Parts of Title 24

| Part | Abbrev. | Code | Model base | Primary adopting agency |
|------|---------|------|------------|-------------------------|
| 1 | CBSC (CAC) | California Administrative Code | n/a (state rule) | BSC |
| 2 | CBC | California Building Code (Vol 1 & 2) | IBC | HCD + BSC + DSA-SS + OSHPD |
| 2.5 | CRC | California Residential Code | IRC | HCD |
| 3 | CEC* | California Electrical Code | NFPA 70 (NEC) | BSC |
| 4 | CMC | California Mechanical Code | UMC (IAPMO) | HCD + BSC |
| 5 | CPC | California Plumbing Code | UPC (IAPMO) | HCD + BSC |
| 6 | Energy Code (Part 6) | California Energy Code | (state-written) | California Energy Commission |
| 7 | (vacant) | — | — | — |
| 8 | CHBC | California Historical Building Code | state-written | SHBSB |
| 9 | CFC | California Fire Code | IFC | SFM |
| 10 | CEBC | California Existing Building Code | IEBC | HCD + BSC |
| 11 | CALGreen | California Green Building Standards Code | state-written | BSC + HCD |
| 12 | — | California Referenced Standards Code | compilation | BSC |

\* Note: "CEC" is sometimes ambiguous — in energy contexts it can mean California **Energy** Commission; in code contexts it means California **Electrical** Code (Part 3). Use "Part 6" or "Title 24 Energy" for the energy code.

## 3. Code Cycle Calendar

California runs a **triennial** adoption cycle. Each cycle publishes July 1 and takes effect six months later on Jan 1 of the next year.

| Cycle | Published | Effective | Status |
|-------|-----------|-----------|--------|
| 2019 | Jul 1, 2019 | Jan 1, 2020 | Superseded |
| **2022** | **Jul 1, 2022** | **Jan 1, 2023** | **CURRENT (through Dec 31, 2025)** |
| **2025** | **Jul 1, 2025** | **Jan 1, 2026** | Takes effect for permits applied for on/after Jan 1, 2026 |

Supplements may be published mid-cycle (e.g., the 2022 CALGreen Supplement took effect July 1, 2024 for Tier 2 measures).

**Transition rule:** Most jurisdictions key compliance to the **permit application submittal date** (not issuance). Projects submitted in Dec 2025 generally fall under the 2022 code even if issued in 2026 — verify with the local jurisdiction.

## 4. Anatomy of a California Code Section

A California Part 2 (CBC) section reads like:

> **CBC 705.8.1 — Allowable Area of Openings** … *[model IBC text]* … **[HCD 1]** … *[HCD amendment]* … **[SFM]** … *[SFM amendment]*

- **Bracketed agency tags** (e.g., `[HCD 1]`, `[SFM]`, `[DSA-SS]`, `[OSHPD 1]`) indicate which state agency wrote that paragraph. Multiple agencies can amend the same section.
- **Matrix Adoption Tables** at the start of each chapter show which agency adopts which section — if no agency adopts a section, it does not apply to that occupancy.
- The **Matrix** is the first place a plan-checker should look when occupancy enforcement is ambiguous.

## 5. State Code vs Local Amendments

A California permit is reviewed against **state code + local amendments**:

1. **State floor.** Title 24 is the minimum. Local governments may amend but must find local climatic, topographic, or geological conditions justifying the amendment and file the findings with BSC (H&SC 17958.7, 18941.5).
2. **Local amendments.** Typically published as a "Building Code Amendments" chapter in the municipal code (e.g., City of San Diego Municipal Code §145.0101+, County of San Diego Code Title 9).
3. **Fire code amendments** are particularly common (zero-lot-line provisions, WUI overlays, NFPA 13D garage sprinklers).
4. **State preemption.** Some state laws override local amendments entirely — e.g., ADU law (Gov Code 65852.2), density bonus (Gov Code 65915), SB 9 (Gov Code 65852.21, 66411.7).

## 6. How to Cite a California Code Section

- **CBC 705.5** → California Building Code (Part 2) Section 705.5
- **CRC R302.1** → California Residential Code (Part 2.5) Section R302.1 (R-prefix retained from IRC)
- **CFC Table 314.4(1)** → California Fire Code (Part 9) Table 314.4(1)
- **CEnC §150.1(c)** or **Title 24 Part 6 §150.1(c)** → California Energy Code
- **CALGreen 4.408.2** → California Green Building Standards Code (Part 11) §4.408.2
- **CPC 411** → California Plumbing Code §411

Always cite the **cycle** when the code text differs (e.g., "2022 CBC 705.8" vs "2025 CBC 705.8"). The section number may be the same but the adoption year changes the wording.
