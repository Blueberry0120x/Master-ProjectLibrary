Overview

This proposal presents a redesigned building organization for 4335 Euclid Avenue. The key change is the elimination of a shared interior hallway in favor of individual private entries for every residential unit — both ground-floor and upper-floor. Each pair of stacked units is physically separated from all neighboring units by a full-height fire wall, creating self-contained residential unit blocks.

This organization unlocks a lighter fire sprinkler system (residential NFPA 13D instead of the commercial-grade NFPA 13R), reduces ongoing maintenance obligations, and produces a building that feels more like individual townhomes than a conventional apartment complex.

✓
Key outcome: No shared hallway. No shared stairwell. Every unit has its own front door at street level. Each unit — including upper-floor units — is entered independently from the exterior. Residents never pass through another unit's space to reach their home.

ℹ
What is a "unit block"? A unit block is one self-contained section of the building enclosed on both sides by full-height fire walls. It contains exactly two homes: one on the ground floor and one directly above it. Each home has its own private front door at street level. The ground-floor unit enters directly; the upper unit enters through its own enclosed private stairwell. No resident of one block can access another block — they are structurally and fire-separated from their neighbors.

Key Benefits

🐘
Lighter Sprinkler System
NFPA 13D (residential-grade) instead of NFPA 13R (commercial-grade). Smaller pipe sizes, fewer heads, lower installation cost, lower maintenance burden.
🏠
Townhome Feel
Every resident has their own front door at street level. No apartment hallways, no buzzers, no neighbors in a shared corridor. A more private, higher-quality living experience.
🔒
Full Fire Separation
Full-height fire walls (CBC 706) from foundation to roof between every unit block. Each block is structurally and fire-independently designed. Exceeds typical apartment construction.
📈
Simpler Code Path
R-3 classification removes the multifamily sprinkler mandate (CBC 903.2.8). Fewer shared-element requirements simplify the permit drawings and reduce plan check exposure.
⚙
Efficient Use of Space
The former 6 ft shared hallway is absorbed into the private stair enclosures. Net usable area loss per unit is minimal — the space works harder with private use than as a shared corridor.
🏛
Commercial Block Retained
The 640 SF ground-floor commercial space (Group B) is retained and fire-separated from the residential blocks. Mixed-use program is preserved as originally planned.

Code Basis (Summary)

The following code sections form the basis for the R-3 classification and sprinkler system selection. Full code analysis will be submitted on the permit drawings.

Code Section Subject Application Result
CBC 706 Fire Walls Full-height fire walls, structurally independent, foundation through roof sheathing, between every unit block ✓ Each block = separate building
CBC 310.5 Group R-3 Classification Buildings with not more than 2 dwelling units — each fire-wall-separated block contains exactly 2 units ✓ R-3 applies to each block
CBC 903.2.8 NFPA 13R Sprinkler Mandate Applies to Group R-2 only — not triggered for R-3 ✓ NFPA 13R not required
CRC R313 Residential Sprinkler Requirement New residential construction requires automatic sprinkler system — NFPA 13D acceptable for R-3 ✓ NFPA 13D required (lighter system)
CBC 508.4 B / R-3 Occupancy Separation Fire-rated separation between commercial block (Group B) and residential blocks (Group R-3) ✓ Shown at commercial/residential interface
⚠
Note on Sprinklers NFPA 13D is required — it is not eliminated entirely. New residential construction in California requires some form of automatic sprinkler system. The benefit of this design is upgrading from the more expensive NFPA 13R (commercial-grade, full-building system) to NFPA 13D (residential-grade, per-unit system), which is smaller in scope, cheaper to install, and simpler to maintain.
