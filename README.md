# Master-ProjectLibrary (MPL)

Consolidated site planning + permit prep monorepo.

**Aliases:** MPL · ProjectLibrary · ProjectBook · PlanCheck

Combines:
- **app/** — interactive parcel map and building configurator (JavaScript/Leaflet)
- **engine/** — permit form filler and plan check agents (Python/PyMuPDF/Claude)
- **data/sites/** — one canonical JSON per project, shared by both layers

## Legacy

Migrated from `ProjectBook-Planner` (design) + `PlanCheck-Planner` (permit).
Both legacy repos archived. This is the active codebase.
