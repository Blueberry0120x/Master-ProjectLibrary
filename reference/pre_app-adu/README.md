# California Pre-Approved ADU Plans — Reference Library

**Legal basis:** AB 1332 (2023), Gov. Code §65852.27 — cities must establish pre-approved ADU plan
programs; applications using a pre-approved plan must receive a decision within **30 days**.
Note: sometimes referenced as "AB 206" but the operative law is AB 1332.

**Downloaded:** 2026-04-03 (updated 2026-04-02)  
**Total files:** 64 PDFs + 6 DWG (AutoCAD) + 6 ZIP plan sets across 11 jurisdictions  
**Folder naming:** merged city+county per metro — city files suffixed `-cty`, county files suffixed `-cou`

**Del Mar Source:** https://www.delmar.ca.us/815/ADU-Sample-Floor-Plans  
**Note:** Labeled "sample floor plans for public use" — still require formal construction drawings. Designed by Stephen Dalton Architects.

**Note:** DWG URL pattern: `{pdsXXX}_acad.dwg` (e.g. `pds670_acad.dwg`). Plans ~85% complete; site-specific info must be added.

**All other jurisdictions** (LA, Sacramento, OC, Fresno, Riverside, Oakland, etc.) distribute **PDF only** — no DWG/DXF/RVT found after exhaustive page inspection.  
**Santa Ana** distributes ZIP archives containing PDFs (no CAD inside).
